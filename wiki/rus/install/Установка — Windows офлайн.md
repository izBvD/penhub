# Установка — Windows офлайн

Установка Python и всех зависимостей сервера на Windows **без интернета** — для изолированной/закрытой машины. Всё необходимое уже собрано в папке [`offline_windows_installer/`](../../../offline_windows_installer) в репозитории.

---

## Что в комплекте

- `install.bat` — установщик (запуск двойным кликом или из консоли).
- `python-3.12.10-amd64.exe` — установщик Python (используется, только если Python в системе ещё нет).
- `packages/` — wheel-файлы всех зависимостей: `fastapi`, `uvicorn`, `openpyxl`, `python-multipart` и их транзитивные пакеты.
- `install_fallback.py` — запасной установщик на случай, если pip не сработает.

---

## Установка

Скопируйте на целевую машину весь проект (вместе с папкой `offline_windows_installer/`) и запустите:

```bat
cd offline_windows_installer
install.bat
```

`install.bat` по шагам:

1. Ищет Python в системе; если не находит — ставит **Python 3.12.10** из вложенного `.exe` (сперва пользовательская установка без прав администратора, при неудаче — системная).
2. Ставит зависимости из локальной папки `packages/` командой `pip --no-index --find-links packages` — **без обращения в интернет**.
3. Если pip не отработал — распаковывает wheel-файлы напрямую через `install_fallback.py` (несовместимые с вашей версией Python wheel'ы пропускаются автоматически).
4. Проверяет, что `fastapi` / `uvicorn` / `openpyxl` / `multipart` / `pydantic` импортируются, и печатает их версии.

> Если Python пришлось установить — **перезапустите терминал** (обновится PATH). После этого запускайте сервер как обычно (см. **[Установка — Сервер](Установка%20—%20Сервер.md)**):
> ```
> python server.py --host 0.0.0.0 --port 322 --password "ВыберитеНадёжныйПароль"
> ```

---

## Если что-то пошло не так

- **Python installation failed** — запустите `install.bat` от имени администратора.
- **Нет `python-3.12.10-amd64.exe`** — скачайте его с [python.org](https://www.python.org/ftp/python/3.12.10/python-3.12.10-amd64.exe) на машине с интернетом, положите рядом с `install.bat`, затем перенесите папку.
- **Уже установлен другой Python** (например 3.14) — установщик его подхватит; нужные под вашу версию wheel'ы выберутся автоматически.
